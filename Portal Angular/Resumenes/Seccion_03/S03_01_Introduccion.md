# Leccion 01 - Introduccion a la Seccion

---

## 1. Titulo de la leccion

**Introduccion a la Seccion 3: JavaScript y TypeScript**

---

## 2. Que se aprende en esta leccion

Esta leccion es una presentacion breve de lo que se va a trabajar en toda la seccion. El instructor explica el objetivo general: reforzar conceptos de JavaScript e introducir TypeScript desde cero, preparando al alumno para trabajar con Angular en secciones posteriores.

---

## 3. Puntos clave explicados

- **Punto de partida:** Se asume que el alumno tiene nociones basicas de JavaScript pero ningun conocimiento previo de TypeScript.
- **Para quien es esta seccion:** Especialmente recomendada para quienes nunca han trabajado con TypeScript. Si ya lo conocen, pueden saltarla e ir directamente a la seccion 4.
- **Que herramienta se usa:** Los ejercicios no se haran directamente en Angular, sino usando una herramienta llamada **Vite** con TypeScript puro (Vanilla TypeScript).
- **Objetivo de la seccion:** Dar un reforzamiento completo de los conceptos que se necesitaran en Angular, reduciendo la curva de aprendizaje cuando se empiece a trabajar con ese framework.

---

## 4. Ejemplo sencillo

En esta leccion no se muestra un ejemplo practico explicito. Es una leccion introductoria que describe el contenido de la seccion.

---

## 5. Funciones, palabras clave o elementos importantes

| Termino | Descripcion |
|---|---|
| **TypeScript** | Lenguaje de programacion que se aprende en esta seccion, partiendo desde cero |
| **JavaScript** | Lenguaje base que el alumno ya conoce; TypeScript lo complementa y extiende |
| **Vite** | Herramienta con la que se trabajaran los ejercicios de esta seccion |
| **Vanilla TypeScript** | TypeScript sin ningun framework adicional, usado para aprender los fundamentos |
| **Seccion 4** | Seccion donde comienza el trabajo real con Angular, despues de este reforzamiento |

---

## 6. Resumen final en pocas palabras

Esta leccion es una introduccion que explica el proposito de la seccion: aprender TypeScript desde cero y reforzar JavaScript para prepararse para Angular. No hay codigo en esta clase, solo una guia de lo que viene y para quien esta dirigido el contenido.
