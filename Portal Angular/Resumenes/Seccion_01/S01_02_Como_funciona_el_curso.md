# Leccion 2: Como Funciona el Curso

---

## ¿Que se aprende en esta leccion?

Esta leccion explica la estructura general del curso: en que orden estan las secciones, cuales son obligatorias y cuales son opcionales, y como sacarle el maximo provecho al material. Tambien se recomienda una hoja de atajos como herramienta de repaso constante.

---

## Puntos clave explicados

- **Seccion 1: Introduccion**
  La seccion actual. Sirve para conocer el curso y prepararse antes de comenzar.

- **Seccion 2: Instalaciones y conceptos basicos**
  Es muy importante no saltarsela. Incluye instalaciones obligatorias y recomendadas, la hoja de atajos, y un video sobre mitos y realidades de Angular.

- **Hoja de atajos**
  Es un documento creado por el instructor con los conceptos mas importantes de Angular. Se recomienda revisarla cada vez que se retome el curso para reforzar lo aprendido.

- **Seccion 3: TypeScript**
  Seccion recomendada para quienes nunca han trabajado con TypeScript. El instructor explica que aprender Angular y TypeScript al mismo tiempo puede ser muy dificil, por eso existe esta seccion separada.

- **Seccion 4 en adelante: Angular propiamente**
  A partir de esta seccion empieza el contenido de Angular moderno, incluyendo "Senales" y "Zoneless Angular".

- **Estructura de secciones en trio**
  Varias aplicaciones se construyen en tres secciones: la primera crea la app, la segunda la expande, y la tercera agrega conceptos avanzados (esta ultima es opcional).

- **Codigo fuente disponible**
  Al final de cada seccion se entrega el codigo tal como quedo, para que el estudiante pueda compararlo si algo no le funciona.

- **Aplicacion administrativa final**
  El curso termina con una aplicacion grande que cubre muchos temas de Angular moderno.

## Funciones, palabras clave o elementos importantes

| Termino | Descripcion |
|---|---|
| **TypeScript** | Lenguaje de programacion que usa Angular. Es una version mejorada de JavaScript con tipado de datos. |
| **Senales (Signals)** | Forma moderna de Angular para gestionar y actualizar datos en pantalla. |
| **Zoneless Angular** | Nuevo modo de Angular que elimina una dependencia interna llamada "Zone.js", haciendolo mas eficiente. |
| **Pipes** | Herramienta de Angular para transformar y dar formato a los datos que se muestran en pantalla. |
| **Codigo fuente** | Archivos con el codigo completo de cada proyecto del curso, disponibles para descarga. |
| **ECMAScript** | Estandar en el que se basa JavaScript. El curso tambien cubre conceptos nuevos de esta especificacion. |

---

## Resumen final en pocas palabras

Esta leccion presenta el mapa completo del curso y explica como aprovecharlo mejor. Se aconseja seguir las secciones en orden, usar la hoja de atajos como referencia permanente, y no saltarse la seccion de TypeScript si no se tiene experiencia previa con ese lenguaje.
