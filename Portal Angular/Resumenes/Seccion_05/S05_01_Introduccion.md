# Seccion 5 - Leccion 01: Introduccion

---

## 1. Titulo de la leccion

**Bienvenida a la Seccion 5: diseno condicional, servicios y despliegue**

---

## 2. Que se aprende en esta leccion

El instructor presenta los temas de la seccion: diseno condicional, nuevo control flow de Angular, directivas, Local Storage, servicios, efectos y despliegue de la aplicacion a produccion.

---

## 3. Puntos clave explicados

- Se expandira lo aprendido en la Seccion 4 con conceptos mas avanzados.
- Se aprende a mostrar y ocultar elementos de forma condicional en el template.
- Se introduce el nuevo control flow de Angular (`@if`, `@for`).
- Se aprende a usar servicios para centralizar el estado de la aplicacion.
- Se exploran los efectos de Angular para sincronizar datos con el Local Storage.
- Se despliega la aplicacion en la web para que cualquiera pueda verla.
- Se advierte que los efectos de Angular tienen casos de uso especificos y no deben usarse para todo.

---

## 4. Ejemplo sencillo

No aplica para esta leccion introductoria.

---

## 5. Funciones, palabras clave o elementos importantes

| Termino | Descripcion |
|---|---|
| `@if` / `@for` | Nuevo control flow de Angular para condiciones e iteraciones |
| Servicios | Clases que centralizan el estado y la logica de negocio |
| `effect()` | Funcion de Angular para ejecutar acciones secundarias cuando una senal cambia |
| Local Storage | Almacenamiento del navegador para persistir datos entre recargas |
| Despliegue | Proceso de subir la aplicacion a un servidor |

---

## 6. Resumen final en pocas palabras

La Seccion 5 expande los fundamentos con diseno condicional, iteraciones, servicios, efectos con Local Storage y despliegue. Los efectos tienen usos especificos y no son una solucion general.
