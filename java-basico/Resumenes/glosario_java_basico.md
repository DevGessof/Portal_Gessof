# GLOSARIO - JAVA BÁSICO
# Términos, Funciones y Palabras Clave Esenciales

---

## A

**Argumento**: Valor que se envía a un método al llamarlo.

**Arreglo**: Ver Vector o Matriz.

**Atributo**: Variable que pertenece a una clase y define una característica del objeto.

---

## B

**boolean**: Tipo de dato lógico que solo acepta dos valores: true (verdadero) o false (falso).

**break**: Termina un case en switch o sale de un bucle inmediatamente.

**Bucle**: Estructura que repite código múltiples veces (while, do-while, for).

**byte**: Tipo de dato numérico entero pequeño (rango: -128 a 127).

---

## C

**case**: Define cada opción en una estructura switch.

**char**: Tipo de dato para un solo carácter, se usa con comillas simples ' '.

**Clase**: Plantilla para crear objetos con atributos y métodos.

**Comentario**: Texto que no se ejecuta, sirve para documentar código (// o /* */).

**Compilador**: Programa que traduce código Java a lenguaje que la computadora entiende.

**Concatenación**: Unir textos o valores usando el operador +.

**Condición**: Expresión que se evalúa como verdadera o falsa.

**Conversión de tipos**: Cambiar un dato de un tipo a otro (ejemplo: String a int).

---

## D

**Declaración**: Crear una variable indicando su tipo y nombre.

**default**: Caso en switch que se ejecuta si ningún case coincide.

**do-while**: Bucle que ejecuta código al menos una vez antes de evaluar la condición.

**double**: Tipo de dato numérico decimal de alta precisión.

---

## E

**else**: Alternativa que se ejecuta cuando la condición del if es falsa.

**else if**: Evalúa una condición adicional si la anterior fue falsa.

**Estructura cíclica**: Ver Bucle.

**Estructura condicional**: Código que se ejecuta solo si se cumple una condición (if, switch).

---

## F

**false**: Valor booleano que representa falso.

**float**: Tipo de dato numérico decimal, requiere 'f' al final del valor.

**for**: Bucle compacto ideal cuando se sabe cuántas veces repetir el código.

---

## I

**IDE**: Entorno de desarrollo integrado (NetBeans, Eclipse).

**if**: Estructura condicional que evalúa si algo es verdadero para ejecutar código.

**import**: Permite usar clases de otros archivos o paquetes.

**Incremento (++)**: Aumenta una variable en 1.

**Índice**: Número que indica la posición en un vector o matriz (comienza en 0).

**Inicialización**: Asignar un valor inicial a una variable.

**Instanciar**: Crear un objeto a partir de una clase usando new.

**int**: Tipo de dato numérico entero estándar.

**Integer**: Versión objeto (wrapper) del tipo int, tiene métodos como parseInt().

---

## J

**JDK**: Kit de desarrollo de Java que incluye el compilador y herramientas.

---

## L

**Librería**: Conjunto de código predefinido que se puede importar y usar.

**Llaves { }**: Delimitan bloques de código en métodos, clases, if, bucles, etc.

---

## M

**main()**: Método principal desde donde inicia la ejecución del programa.

**Matriz**: Arreglo bidimensional con filas y columnas [fila][columna].

**Método**: Bloque de código reutilizable que realiza una tarea específica.

---

## N

**new**: Palabra clave para crear nuevos objetos o arreglos en memoria.

**nextInt()**: Método de Scanner para capturar números enteros.

**nextLine()**: Método de Scanner para capturar texto completo.

**nextDouble()**: Método de Scanner para capturar números decimales.

---

## O

**Objeto**: Instancia de una clase con valores específicos.

**Operador**: Símbolo que realiza operaciones (+, -, *, /, ==, <, >, etc.).

---

## P

**package**: Define a qué paquete pertenece un archivo Java.

**Parámetro**: Variable que recibe un método para trabajar con ella.

**parse**: Convertir un String a otro tipo de dato (parseInt, parseDouble, etc.).

**POO**: Programación Orientada a Objetos, paradigma basado en clases y objetos.

**print()**: Imprime en pantalla sin salto de línea.

**println()**: Imprime en pantalla con salto de línea al final.

**Punto y coma (;)**: Termina cada instrucción en Java.

---

## R

**return**: Devuelve un valor desde un método y termina su ejecución.

---

## S

**Scanner**: Clase para capturar datos ingresados por teclado.

**static**: Palabra clave que permite usar métodos sin crear objetos.

**String**: Tipo de dato para cadenas de texto, se usa con comillas dobles " ".

**switch**: Estructura condicional por selección para múltiples casos específicos.

**System.in**: Entrada estándar (teclado).

**System.out.println()**: Comando para imprimir en pantalla con salto de línea.

---

## T

**true**: Valor booleano que representa verdadero.

**Tipo de dato**: Especifica qué clase de información almacena una variable (int, String, double, etc.).

---

## V

**Variable**: Espacio en memoria con nombre que almacena un valor.

**Vector**: Arreglo unidimensional que almacena múltiples valores del mismo tipo.

**void**: Indica que un método no devuelve ningún valor.

---

## W

**while**: Bucle que repite código mientras una condición sea verdadera.

**Wrapper**: Clase que envuelve un tipo primitivo (Integer, Double, Boolean, etc.).

---

## SÍMBOLOS Y OPERADORES

**=**: Asignación, guarda un valor en una variable.

**==**: Comparación de igualdad, verifica si dos valores son iguales.

**!=**: Comparación de desigualdad, verifica si dos valores son diferentes.

**<**: Menor que.

**>**: Mayor que.

**<=**: Menor o igual que.

**>=**: Mayor o igual que.

**+**: Suma o concatenación.

**-**: Resta.

**\***: Multiplicación.

**/**: División.

**%**: Módulo, devuelve el residuo de una división.

**++**: Incremento en 1.

**--**: Decremento en 1.

**&&**: AND lógico, verdadero si ambas condiciones son verdaderas.

**||**: OR lógico, verdadero si al menos una condición es verdadera.

**!**: NOT lógico, invierte el valor booleano.

**[]**: Corchetes para declarar y acceder a vectores y matrices.

**.**: Operador punto para acceder a atributos y métodos de objetos.

**//**: Comentario de una línea.

**/\* \*/**: Comentario de múltiples líneas.

---

## MÉTODOS COMUNES

**Integer.parseInt(String)**: Convierte String a int.

**Double.parseDouble(String)**: Convierte String a double.

**Float.parseFloat(String)**: Convierte String a float.

**Byte.parseByte(String)**: Convierte String a byte.

**entrada.nextInt()**: Captura un número entero del teclado.

**entrada.nextDouble()**: Captura un número decimal del teclado.

**entrada.nextLine()**: Captura texto completo del teclado.

**System.out.println()**: Imprime texto en pantalla con salto de línea.

**System.out.print()**: Imprime texto en pantalla sin salto de línea.

---

## CONCEPTOS CLAVE

**Sintaxis**: Reglas de escritura del lenguaje Java.

**Compilación**: Proceso de convertir código Java a bytecode ejecutable.

**Ejecutar**: Correr el programa para ver sus resultados.

**Debugging**: Proceso de encontrar y corregir errores en el código.

**OpenJDK**: Versión gratuita del Kit de Desarrollo de Java.

**NetBeans**: IDE popular para desarrollo en Java.

**Proyecto**: Contenedor que agrupa todos los archivos de una aplicación Java.

**Package**: Carpeta que organiza clases relacionadas.

**Java Class**: Archivo .java que contiene una clase.

---

## ESTRUCTURAS FUNDAMENTALES

**Secuencial**: Código que se ejecuta línea por línea de arriba hacia abajo.

**Condicional**: Código que se ejecuta solo si se cumple una condición (if, switch).

**Cíclica**: Código que se repite múltiples veces (while, do-while, for).

---

Este glosario cubre los términos esenciales del curso básico de Java. Úsalo como referencia rápida mientras programas.
