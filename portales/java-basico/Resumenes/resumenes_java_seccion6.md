## LECCIÓN 19: PERSONALIZAR EL IDE APACHE NETBEANS - DRACULA THEME

### ¿Qué se aprende en esta lección?

En esta lección se aprende a personalizar la apariencia del IDE Apache NetBeans instalando un tema oscuro llamado Dracula. Se explica cómo descargar, instalar y configurar el tema para hacer más cómodo y agradable el trabajo de programación, especialmente para reducir el cansancio visual.

### Puntos clave explicados

- **Importancia de la comodidad**: Un entorno de trabajo cómodo mejora la experiencia de aprendizaje.

- **Temas oscuros**: Reducen el brillo de la pantalla y son más cómodos para los ojos durante sesiones largas de trabajo.

- **Dracula Theme**: Tema popular y profesional disponible para NetBeans, similar a los temas de otros IDE como Visual Studio.

- **Proceso de instalación**: 
  1. Buscar y descargar el tema
  2. Importarlo en NetBeans
  3. Ajustar el tamaño de fuente
  4. Personalizar según preferencias

- **Configuración adicional**: Se puede cambiar el tipo de fuente, tamaño y otros aspectos visuales.

- **Atajo útil**: Alt + rueda del mouse permite ampliar o reducir el tamaño del texto mientras se trabaja.

### Ejemplo sencillo

Pasos detallados para instalar el tema:

**Paso 1 - Descargar el tema:**
1. Abrir el navegador web
2. Buscar en Google: "Dracula theme Java NetBeans"
3. Entrar a draculatheme.com
4. Buscar la sección de NetBeans
5. Hacer clic en "Download File"
6. Descargar el archivo (no es necesario descomprimirlo)

**Paso 2 - Importar en NetBeans:**
1. Ir a Tools > Options
2. Seleccionar "Fonts and Colors"
3. Crear un duplicado del perfil actual (botón "Duplicate")
4. Darle un nombre (ejemplo: "prueba")
5. Hacer clic en el botón "Import" (parte inferior)
6. Buscar y seleccionar el archivo descargado
7. Hacer clic en "Abrir"
8. Confirmar el reemplazo (OK)
9. Guardar cambios (OK)
10. NetBeans se reiniciará con el nuevo tema

**Paso 3 - Ajustar el tamaño de fuente:**
1. Ir a Tools > Options
2. Seleccionar "Fonts and Colors"
3. En la pestaña "Syntax"
4. Cambiar el tamaño de fuente (ejemplo: 16 o 17)
5. Aplicar cambios (Apply)
6. Guardar (OK)

### Funciones, palabras clave o elementos importantes

- **Dracula Theme**: Tema de colores oscuro popular para editores de código.

- **Tools > Options**: Menú de configuración de NetBeans.

- **Fonts and Colors**: Sección donde se configuran las fuentes y colores del editor.

- **Import**: Botón para importar temas externos al IDE.

- **Duplicate**: Opción para crear una copia del perfil actual antes de hacer cambios.

- **Font**: Tipo de letra usado en el editor de código.

- **Size**: Tamaño de la fuente en el editor.

- **Alt + rueda del mouse**: Atajo para ampliar/reducir texto rápidamente.

- **draculatheme.com**: Sitio web oficial del tema Dracula.

### Resumen final en pocas palabras

Personalizar NetBeans con un tema oscuro como Dracula mejora la comodidad visual al programar. El proceso es simple: descargar el tema, importarlo en NetBeans y ajustar el tamaño de fuente según preferencia. Esta personalización hace más agradable el aprendizaje y reduce el cansancio visual durante sesiones largas de programación.
